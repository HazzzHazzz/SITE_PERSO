Animoclic

Animoclic est un jeu que j'ai développé en JavaScript et en utilisant jQuery. Dans ce jeu pour enfant, le joueur doit cliquer sur la carte correspondant à l'indice donné. Mon code JS n'est, à mon sens, pas très bien optimisé à plusieurs niveaux :

  • Le code est plutôt répétitif au niveau du Switch
  • Difficilement maintenable du fait de l'utilisation d'ID

Cependant, c'est exactement la raison pour laquelle je développe ce jeu, mon besoin d'entrainement. J'ai du répondre à plusieurs intérogation afin de développer ce jeu :

  • Comment générer un indice aléatoire ? (Utilisation de Math.random)
  • Comment générer une boucle du type : tant que les images n'ont pas toutes été cliquées, je continue d'envoyer des consignes, sinon, j'affiche le mot victoire (Utilisation d'un Switch)
  • Une fois qu'une image a été cliquée, comment "supprimer" l'indice qui lui correspond afin qu'il ne réapparaisse pas ? (Utilisation d'un tableau et de .splice)

En bref, un terrain de jeu concret sur lequel je ne peux que m'améliorer.
